# Olympic Games Informative Website

## 📌 Description
The Olympic Games Informative Website is a responsive web application developed to provide users with comprehensive information about the Olympic Games. It includes details about sports, events, schedules, and news. The project focuses on user-friendly design, easy navigation, and scalability for future enhancements.

## 🚀 Features
- Responsive design for mobile and desktop
- Home page with Olympic overview
- About Us section
- Sports information pages
- News updates section
- Timetable/Schedule page
- Contact page
- Sign-in functionality with JavaScript validation
- Structured navigation using header and footer

## 🛠 Tech Stack
- HTML
- CSS
- Bootstrap
- JavaScript
- PHP (for backend structure)


## ▶️ How to Run the Project
1. Install XAMPP
2. Move the project folder to: C:\xampp\htdocs
3. Start Apache from XAMPP Control Panel
4. Open browser and run: http://localhost/flagbit_solution/
